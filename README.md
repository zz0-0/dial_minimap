## Run Locally

Clone the project

```bash
git clone https://github.com/zz0-0/dial_minimap
```

Go to the project directory

```bash
cd dial_minimap
```

Install dependencies

```bash
flutter pub get
```

Run the project

- Option 1: VSCode

```bash
cd example/lib
```

- main.dart
  - click Run
